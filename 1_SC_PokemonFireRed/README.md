### 📌 실행 전 필수 사항

- `Game1` 프로젝트를 **시작 프로젝트로 설정**해야 합니다.
- `갈무리 폰트` 3가지를 **모두 설치**해야 합니다.


### 🎮 게임 조작법 안내

| 동작     | 키 입력           |
|:---------|:------------------|
| 이동     | `방향키`          |
| 선택     | `Z`               |
| 취소     | `X`               |
| 메뉴     | `Enter`           |
| 길 찾기  | `A`               |
