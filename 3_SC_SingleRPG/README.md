### 🖱️ 게임 조작법 안내

| 동작             | 키 입력 또는 조작 방식                         |
|:------------------|:---------------------------------------------|
| 이동             | `우클릭`                                      |
| 선택             | `좌클릭`                                      |
| 기본 공격        | `A`                                           |
| 스킬 공격        | `Q`, `W`, `E`                                 |
| 인벤토리         | `I`                                           |
| 포션 사용        | `D`, `F` 또는 인벤토리에서 더블 클릭          |
| 카메라 회전      | 마우스 휠 클릭 상태에서 마우스 이동           |
| 카메라 이동      | 화면 끝자락으로 마우스 이동                  |
| 카메라 거리 조절 | 마우스 휠                                     |
| 탱커 선택        | `1`                                           |
| 딜러 선택        | `2`                                           |
| 힐러 선택        | `3`                                           |
| 모두 선택        | `4`                                           |
