### 🎮 게임 조작법 안내

| 동작         | 키 입력         |
|:-------------|:----------------|
| 이동         | `방향키`        |
| 공격         | `A`             |
| 점프         | `C`             |
| 더블 점프     | `C` 두 번       |
| 윗 점프       | `↑` + `C`       |
