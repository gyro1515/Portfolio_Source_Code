### 📌 실행 전 필수 사항

- `Font` 폴더에 있는 폰트를 **모두 설치**해야 정상적으로 출력됩니다.


### 🎮 게임 조작법 안내

| 동작                         | 키 입력 또는 조작 방식                     |
|:------------------------------|:------------------------------------------|
| 이동                         | `W`, `A`, `S`, `D`                        |
| 뛰기                         | `Left Shift`                             |
| 점프                         | `Spacebar`                               |
| 인벤토리                     | `U`                                      |
| 플레이어 팰 리스트           | `P`                                      |
| 팰 소환                      | `E`                                      |
| 소환할 팰 바꾸기             | `1`, `3`                                 |
| 팰 동작 바꾸기               | `4`                                      |
| 포획 스피어 투척             | `Q`                                      |
| 건축물 보기                  | `B`                                      |
| 건축물과 상호작용 / 건축     | `F`                                      |
| 상호작용 끄기                | `Esc`                                    |
| 줌인                         | `우클릭`                                 |
| 착용한 무기 바꾸기           | 마우스 휠                                 |
| 총 쏘기                      | `좌클릭`                                 |
| 총 장전                      | `R`                                      |
| 마이닝                       | 곡괭이 든 상태에서 `G`                   |
| 아이템 제작대에 아이템 넣기  | `J`                                      |
| 아이템 제작대에서 제작       | `Y`                                      |
| 마우스 보이기                | `Z`                                      |
